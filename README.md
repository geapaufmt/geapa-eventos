# geapa-eventos

Módulo do sistema GEAPA responsável pela gestão de eventos que não são apresentações de membros.

## Escopo inicial
- reuniões gerais
- reuniões de diretoria
- reuniões deliberativas
- palestras com convidados
- minicursos
- treinamentos internos
- visitas técnicas
- eventos extraordinários

## Observação
A gestão de apresentações de membros permanece no módulo `geapa-apresentacoes`.
